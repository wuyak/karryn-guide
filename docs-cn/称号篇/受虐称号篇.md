被动 防御 +10%

使用方式 : 同上

受虐称号 所需被动

前期 选择欲望时 选择 皮肤 跟 M等级 提升 2级

你所要 提升的 被动  
每个 被动 提升 M等级 我只写重点被动  
1.战败的女看守 : 战败后取得  
效果 - 猥谈敏感度根据M等级增加  
  
2.逐渐屈服的内心 : 战败后取得  
  
3.屁股上的手印 : 第五欲望够高发生打屁股攻击取得  
效果 - 被打屁股后容易造成失衡状态  
  
4.异常性癖 : 第五欲望够高发生打屁股攻击取得  
效果 - 打屁股敏感度根据勇气值提升  
  
5.崩坏的自尊心 : 失衡状态下取得  
效果 - 学会 顺势休息  
  
6.猥谈100% : 猥谈攻击 100% 取得  
  
7.故意耗尽的体力 : 使用放弃 取得  
效果 - 自我防卫时100%无法使用重新振作  
  
8.吸食手指的女看守 : 第二欲望高时囚犯使用手指取得  
效果 - 在同一天内每吸食囚犯手指就增加初始第一欲望

\----------

这边开始是危险分支 需要部位动作

请注意是 主动使用 而不是 被动使用  
1.抛弃羞耻的舔工 : 使用舔部字体取得  
效果 - 在同一天每使用舔部字体后嘴唇敏感度将提升  
  
2.舔工成瘾的抖M : 舔部字体满级学会取得  
效果 - 勇气值低时 舔部字体释放压力量提升  
  
3.压力释放器 : 第二层战败取得  
  
4.人生拷问的快感 : 第三层or第四层战败取得  
  
5.压力量是美容的秘密 : 释放压力量在身上取得  
效果 - 释放压力量在身上后容易自我防卫  
永久效果 - 压力量有机率不被清除 (睡觉后将清除)

隐藏 被动 未列入 受虐被动中 但一样会提升 受虐等级  
1.屈服的快感 : 第三层or第四层战败取得  
效果 - 在地下室闪避率提升  
  
2.快乐时光的人生拷问 : 第三层or第四层战败取得  
效果 - 隔离房事件后睡眠品质提升
