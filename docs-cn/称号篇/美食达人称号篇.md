基数菜单  
1.镶银芽  
食用效果 : 经验值 +50%  
使用方式 : 升级用 后期不需要

2.佛跳墙  
食用效果 : 体力&精力&意志力 各恢复 +5%  
使用方式 : 想恢复量提升就使用 特别是 SE 可以用到  
  
3.脆皮烤乳猪  
食用效果 : 体力 +15% & 100%时减少精力消耗  
使用方式 : SE 也可以用到  
  
4.珍珠丸子  
食用效果 : 每回合开始前 兴奋状态机率提升 取代 自信状态  
使用方式 : 想快速提升欲望 使用  
  
5.蒜蓉粉丝蒸鲑鱼  
食用效果 : 山部 与 洞部 提升 敏感度 +50%  
使用方式 : 用于特定 人生技能使用  
\--------------------  
偶数菜单  
1.百屋朝凤  
食用效果 : 力量&灵巧&敏捷 各 +5%  
使用方式 : 基本属性提升 实用 但看个人需求  
  
2.文思豆腐  
食用效果 : 疲劳积累 -33%  
使用方式 : 后期压制 楼层 用 光一场就要 8 ~ 到20 疲劳  
  
3.素熊掌  
食用效果 : 触碰耐性 & 人生耐性 各 +15%  
使用方式 : 想增加耐性使用 欲望少于 50 以下 可不用吃  
  
4.樱桃肉  
食用效果 : 敌人攻击性 下降 包括 囚犯 & 暴徒 (暴动) & 卫兵  
使用方式 : 想要刷 触碰 & 视奸 被动 使用  
可装备 [魅魔典狱长](./魅力称号篇.md) 更实用  
  
5.松鼠鲑鱼  
食用效果 : 口部 与 臂部 敏感度 +50%  
使用方式 : 用于特定 人生技能使用
