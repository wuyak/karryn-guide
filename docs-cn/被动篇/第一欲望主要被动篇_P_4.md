前言 :

此篇章为 获得教学 与 被动内容 取得教学

台湾自制攻略 & Taiwan homemade Raiders

\--------------------

请注意 被动顺序依本人获得为主  
61.失去月工破女  
获得方式 : 初次月工字体取得  
被动效果 : 力量 提升 & 猥谈耐性 下降  
被动正效果 : 第一欲望获得率 下降  
  
62.三口人生初体验  
获得方式 : 初次三个字体时取得  
被动效果 : 三个字体时 人生耐性下降  
被动反效果 : 同一天中 每三个字体时 开场第一欲望提升  
  
63.初尝败北滋味  
获得方式 : 战败取得  
被动效果 : 第一欲望上限 提升 & 猥谈耐性 下降  
被动反效果 : 猥谈敏感度 提升  
  
64.饥渴的监狱魅魔  
获得方式 : 1.连续字体取得 2.兴奋状态下连续字体取得  
被动效果 : 所有欲望下限 & 所有欲望上限 & 所有欲望获得率 提升  
  
65.月工中山的快感  
获得方式 : 月工字体中山取得  
被动效果 : 第一与第五欲望上限 提升  
被动反效果 : 月工中山敏感度根据勇气值提升  
  
66.被压力灌满的月工  
获得方式 : 月工字体中山取得  
被动效果 : 第一与第五欲望上限 提升  
被动反效果 : 月工压力量的量 提升  
  
67.中山100%  
获得方式 : 洞部字体时 100%取得  
被动效果 : 开场第一与第四欲望上限 提升  
被动反效果 : 同一天中 洞口被压力量中山 根据次数增加 洞口中山敏感度  
  
68.露出成瘾  
获得方式 : 晚上巡逻取得  
被动效果 : 魅力提升  
被动反效果 : 晚上巡逻时洞口容易出水  
  
69.为阿津而生的双峰  
获得方式 : 被乳部字体取得  
被动效果 : 第一与第三欲望上限 提升  
新增技能 : 乳部字体  
  
70.月工指导者  
获得方式 : 被月工字体取得  
被动效果 : 第一与第五欲望上限 提升  
新增技能 : 月工字体  
  
71.压力月工100%  
获得方式 : 月工字体被压力量中山100%时取得  
被动效果 : 开场第一与第五欲望上限 提升  
被动反效果 : 同一天中 月工被压力量中山 根据次数增加 月工中山敏感度  
  
72.永远握住阿津  
获得方式 : 使用手部字体取得  
被动效果 : 手部字体等级 & 第一欲望获得率 提升  
被动反效果 : 同一天中 给不同的阿津手部字体 开场第一欲望 提升  
  
73.听见阿津的话题  
获得方式 : 猥谈攻击时取得  
被动效果 : 第一欲望上限 提升  
被动正效果 : 同一场战斗中 根据猥谈次数增加人生耐性  
  
74.对阿津毫无抵抗的洞口  
获得方式 : 被洞部字体取得  
被动效果 : 每回合第一与第五欲望 & 洞口压力量的量 提升  
被动反效果 : 同一天中 每被抽山 将提升洞口中山敏感度  
  
75.母猪的三口天堂  
获得方式 : 数次被三个字体取得  
被动效果 : 开场所有欲望 & 被三个字体时爆击率 提升  
  
76.月工100%  
获得方式 : 月工字体时 100%取得  
被动效果 : 开场第一与第五欲望上限 提升  
被动反效果 : 同一天中 每被月工抽山 将提升月工敏感度  
  
77.失控的下半身  
获得方式 : 数次被洞部&月工字体取得  
被动效果 : 当没被二个字体时 第一欲望自动 提升 & 当被二个字体时 人生耐性 下降  
  
78.监狱人气女典狱长  
获得方式 : 数次使用or被洞部字体取得  
被动效果 : 洞部字体等级 & 第一欲望上限 提升  
被动反效果 : 同一场战斗中 每被洞口抽山 每回合开始第四欲望自动提升  
  
79.松弛的洞口 压力的好去处  
获得方式 : 数次被洞口中山取得  
被动效果 : 洞口敏感度 & 第一与第五欲望获得率 提升  
被动反效果 : 同一天中 每被洞口中山 防御力降低  
  
80.陷入100%地狱的廉价清洁工  
获得方式 : 数次100%取得  
被动效果 : 100%时失神机率 下降  
被动反效果 : 同一天中 每100%时 所有耐性下降
