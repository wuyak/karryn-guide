此篇章为 获得教学 与 被动内容 取得教学

台湾自制攻略 & Taiwan homemade Raiders

\--------------------  
请注意 被动顺序依本人获得为主  
41.触碰派阿津学家  
获得方式 : 使用触碰阿津取得  
被动效果 : 触碰等级 & 第一欲望上限 提升  
被动反效果 : 同一场战斗中 第一欲望获得率根据触碰阿津数量提升  
  
42.吸食手指的女典狱长  
获得方式 : 吸过数次男人手指过  
被动效果 : 受虐等级 提升 > [受虐称号篇](/称号篇/受虐称号篇.md)  
被动反效果 : 同一天中 每吸食手指 当天开场第一欲望 提升  
  
43.触碰100%  
获得方式 : 触碰时100%取得  
被动效果 : 开场所有欲望提升 & 触碰耐性 & 落衣耐性 下降  
  
44.钢管式防卫  
获得方式 : 使用武器自我防卫  
被动效果 : 第一欲望上限 提升  
被动正效果 : 使用武器自我防卫 囚犯兴奋机率提升  
  
45.认津不认人  
获得方式 : 使用触碰阿津取得  
被动效果 : 触碰等级 & 开场第一欲望上限 提升  
被动反效果 : 同一天中 人生耐性根据触碰阿津数量 提升  
  
46.期待已久的初吻  
获得方式 : 初吻没了取得  
被动效果 : 魅力 & 第一欲望上限 提升  
被动反效果 : 亲吻所需的第二欲望降低15  
  
47.初次吸食阿津  
获得方式 : 初次口部字体取得  
被动效果 : 灵巧 & 第一欲望上限 提升  
被动正效果 : 第一欲望获得率 下降  
  
48.饮量心经  
获得方式 : 喝过压力量取得  
被动效果 : 第一与第二欲望上限 & 喝取压力量的量 提升  
  
49.破坏  
获得方式 : 初次洞部字体取得  
被动效果 : 猥谈耐性 下降  
被动正效果 : 第一欲望获得率 下降  
被动反效果 : 人生玩具所需的第四欲望降低20  
  
50.充实的洞口  
获得方式 : 洞口拥有压力量取得  
被动效果 : 第一与第四欲望上限 & 洞口压力量的量 提升  
  
51.月工的情欲  
获得方式 : 第五欲望 100时取得  
被动效果 : 第一欲望上限 提升  
被动反效果 : 第五欲望高于 100时触碰耐性下降  
  
52.多人中山  
获得方式 : 同一场战斗中数次中山取得  
被动效果 : 第一与第四欲望上限 提升  
被动反效果 : 同一场战斗中 中山次数 将提升中山敏感度  
  
53.人生100%  
获得方式 : 洞部字体时 100%取得  
被动效果 : 第一与第四欲望上限 提升  
被动反效果 : 同一场战斗中 每被抽山 将提升洞部敏感度  
  
54.乐意吞量  
获得方式 : 数次喝过压力量取得  
被动效果 : 第一与第二欲望上限 提升  
被动反效果 : 同一场战斗中 每给不同阿津口部字体后 吞量敏感度 提升  
  
55.女典狱长洞口解禁  
获得方式 : 被洞部字体数次取得  
被动效果 : 第一欲望上限 & 第四欲望获得率 提升  
新增技能 : 洞部字体  
  
56.受虐100%  
获得方式 : 受虐时 100%取得 > [受虐称号篇](/称号篇/受虐称号篇.md)  
被动反效果 : 受虐等级根据第一欲望提升 & 战败时所有欲望获得率根据受虐等级提升  
  
57.口部学徒  
获得方式 : 被口部字体数次取得  
被动效果 : 第一欲望上限 & 第二欲望获得率 提升  
新增技能 : 口部字体  
  
58.永不满足的100%  
获得方式 : 数次100%取得  
被动效果 : 每回合随机欲望自动提升 & 100%时失神机率 降低  
被动反效果 : 同一天中 每100%时 降低精力上限  
  
59.人形自走R26  
获得方式 : 任意字体100人取得  
被动效果 : 开场所有欲望提升 & 人生耐性 降低  
新增技能 : 投降  
  
60.初次乳部字体  
获得方式 : 初次乳部字体取得  
被动效果 : 灵巧 & 第一欲望上限 & 第四欲望获得率 提升
