台湾自制攻略 & Taiwan homemade Raiders

\--------------------  
请注意 被动顺序依本人获得为主  
1.禁断入口  
获得方式 : 第一次与囚犯战斗取得  
被动效果 : 每回合随机欲望提升  
  
2.阿津猎人 > [施虐称号篇](/称号篇/施虐称号篇.md)  
获得方式 : 打赢囚犯数场取得  
被动效果 : 施虐等级 & 力量 % 第一欲望上限 提升  
  
3.阿津欺凌专家 > [施虐称号篇](/称号篇/施虐称号篇.md)  
获得方式 : 打赢囚犯数场取得  
被动效果 : 施虐等级 提升 & 当使用胯下踢击时 第一欲望 获取根据 施虐等级提升  
被动反效果 : 同一天中 每次使用胯下踢击 更容易发生 兴奋状态  
  
4.浑身燥热  
获得方式 : 1.保持兴奋模式战斗数场取得 2.第二层BOSS战结束后取得  
被动效果 : 所有欲望上限 提升  
被动反效果 : 兴奋状态 下 所有欲望自动提升  
  
5.初见压力量瞬间  
获得方式 : 第一次看过囚犯释放压力量取得  
被动效果 : 精神 & 第一欲望上限 提升  
  
6.目睹阿津  
获得方式 : 第一次看过囚犯使用阿津取得  
被动效果 : 精神 & 第一欲望上限 提升  
  
7.目睹不可言说之物  
获得方式 : 数次看过囚犯使用阿津取得  
被动效果 : 第一欲望上限 提升  
新增技能 : 凝视阿津  
  
8.变身欲女的征兆  
获得方式 : 首次使用凝视阿津取得  
被动效果 : 第一欲望上限 提升  
技能升级 : 凝视阿津 效果 提升 & 冷却时间 缩短  
  
9.对阿津的好奇心  
获得方式 : 数次使用凝视阿津取得  
被动效果 : 开场第一欲望 提升  
技能升级 :  凝视阿津 效果 提升  
技能反效果 : 在同一场战斗中 每看过一只阿津 第一欲望获得提升  
  
10.观察阿津爱好  
获得方式 : 数次使用凝视阿津取得  
被动效果 : 每回合开始 根据上扬的阿津而提升 开始第一欲望  
技能反效果 : 在同一天中 每看一次阿津 当天开场第一欲望 提升  
  
11.猥谈女主角  
获得方式 : 战斗中听囚犯说话  
被动效果 : 第一欲望上限 提升  
被动正效果 : 在同一天中 每听到一次猥谈 攻击力提升  
  
12.躲藏的视野  
获得方式 : 第一欲望 50时取得  
被动效果 : 第一欲望上限 提升  
被动反效果 : 第一欲望高于50时 攻击力下降  
  
13.初次升天  
获得方式 : 首次100%取得  
被动效果 : 所有欲望上限 提升 & 触碰与人生耐性 下降  
被动反效果 : 在同一场战斗中 每次100%时 精力上限 下降  
  
14.100%余温  
获得方式 : 100%时无法行动取得  
被动效果 : 每回合随机欲望自动提升 & 100%时余温更多  
被动正效果 : 100%时耗费的精力减少  
  
15.被挑拨的好奇心  
获得方式 : 看囚犯使用阿津取得  
被动效果 : 第一欲望上限 提升 & 战斗中自我防卫机率提升 & 囚犯使用阿津时所有欲望上升更快  
  
16.被沾污的美丽肌肤  
获得方式 : 首次被压力量沾到身上取得  
被动效果 : 第一欲望上限 提升  
被动反效果 : 敏捷 根据压力量在身上而 下降  
  
17.沾上压力量的触感  
获得方式 : 数次被压量沾到身上取得  
被动效果 : 第一欲望上限 提升 & 当使用 口&乳&足 字体时 体外释放压力量敏感度提升  
  
18.压力量爱好  
获得方式 : 数次被压量沾到身上取得  
被动效果 : 第一欲望上限 提升  
被动反效果 : 落衣耐性根据身上的压力量而下降  
  
19.被阿津吸引的目光  
获得方式 : 数次看过囚犯使用阿津取得  
被动效果 : 第一欲望上限 提升 & 战斗中自我防卫机率提升 & 囚犯使用阿津时所有欲望上升更快  
  
20.对阿津的美妙幻想  
获得方式 : 数次看过囚犯使用阿津取得  
被动效果 : 兴奋状态时 第一欲望自动提升  
被动反效果 : 第一欲望不是最高时 第一欲望获得提升
