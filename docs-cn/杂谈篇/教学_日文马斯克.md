\--------------------  
解决 日文马斯克 问题  
请打开游戏 Karryn's Prison 资料夹  
选择 www 中的 js 之后点击 plugins  
文件夹 里面的 RemtairyDebugMode  
用 记事本 打开此文件  
往下拉到最下面  
在这行 return JAPAN\_MODE 前面加上 //  
变成 //return JAPAN\_MODE (懒人法直接复制贴上)  
保存之后进入游戏 日文就没有马斯克了  
请用 v 1.0.1 G3 版本的 RemtairyDebugMode 套用到新版本 之前版本也可以喔  
之前 v 1.0.1 H4 版本 已失效 上面方法有效  
最新版本 v 1.0.1 K 一样有效
