受虐稱號 所需被動

前期 選擇慾望時 選擇 皮膚 跟 M等級 提升 2級

你所要 提升的 被動  
每個 被動 提升 M等級 我只寫重點被動  
1.戰敗的女看守 : 戰敗後取得  
效果 - 猥談敏感度根據M等級增加  
  
2.逐漸屈服的內心 : 戰敗後取得  
  
3.屁股上的手印 : 第五慾望夠高發生打屁股攻擊取得  
效果 - 被打屁股後容易造成失衡狀態  
  
4.異常性癖 : 第五慾望夠高發生打屁股攻擊取得  
效果 - 打屁股敏感度根據勇氣值提升  
  
5.崩壞的自尊心 : 失衡狀態下取得  
效果 - 學會 順勢休息  
  
6.猥談100% : 猥談攻擊 100% 取得  
  
7.故意耗盡的體力 : 使用放棄 取得  
效果 - 自我防衛時100%無法使用重新振作  
  
8.吸食手指的女看守 : 第二慾望高時囚犯使用手指取得  
效果 - 在同一天內每吸食囚犯手指就增加初始第一慾望

請注意是 主動使用 而不是 被動使用  
1.拋棄羞恥的舔工 : 使用舔部字體取得  
效果 - 在同一天每使用舔部字體後嘴唇敏感度將提升  
  
2.舔工成癮的抖M : 舔部字體滿級學會取得  
效果 - 勇氣值低時 舔部字體釋放壓力量提升  
  
3.壓力釋放器 : 第二層戰敗取得  
  
4.人生拷問的快感 : 第三層or第四層戰敗取得  
  
5.壓力量是美容的秘密 : 釋放壓力量在身上取得  
效果 - 釋放壓力量在身上後容易自我防衛  
永久效果 - 壓力量有機率不被清除 (睡覺後將清除)
