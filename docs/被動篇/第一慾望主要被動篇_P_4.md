請注意 被動順序依本人獲得為主  
61.失去月工破女  
獲得方式 : 初次月工字體取得  
被動效果 : 力量 提升 & 猥談耐性 下降  
被動正效果 : 第一慾望獲得率 下降  
  
62.三口人生初體驗  
獲得方式 : 初次三個字體時取得  
被動效果 : 三個字體時 人生耐性下降  
被動反效果 : 同一天中 每三個字體時 開場第一慾望提升  
  
63.初嘗敗北滋味  
獲得方式 : 戰敗取得  
被動效果 : 第一慾望上限 提升 & 猥談耐性 下降  
被動反效果 : 猥談敏感度 提升  
  
64.飢渴的監獄魅魔  
獲得方式 : 1.連續字體取得 2.興奮狀態下連續字體取得  
被動效果 : 所有慾望下限 & 所有慾望上限 & 所有慾望獲得率 提升  
  
65.月工中山的快感  
獲得方式 : 月工字體中山取得  
被動效果 : 第一與第五慾望上限 提升  
被動反效果 : 月工中山敏感度根據勇氣值提升  
  
66.被壓力灌滿的月工  
獲得方式 : 月工字體中山取得  
被動效果 : 第一與第五慾望上限 提升  
被動反效果 : 月工壓力量的量 提升  
  
67.中山100%  
獲得方式 : 洞部字體時 100%取得  
被動效果 : 開場第一與第四慾望上限 提升  
被動反效果 : 同一天中 洞口被壓力量中山 根據次數增加 洞口中山敏感度  
  
68.露出成癮  
獲得方式 : 晚上巡邏取得  
被動效果 : 魅力提升  
被動反效果 : 晚上巡邏時洞口容易出水  
  
69.為阿津而生的雙峰  
獲得方式 : 被乳部字體取得  
被動效果 : 第一與第三慾望上限 提升  
新增技能 : 乳部字體  
  
70.月工指導者  
獲得方式 : 被月工字體取得  
被動效果 : 第一與第五慾望上限 提升  
新增技能 : 月工字體  
  
71.壓力月工100%  
獲得方式 : 月工字體被壓力量中山100%時取得  
被動效果 : 開場第一與第五慾望上限 提升  
被動反效果 : 同一天中 月工被壓力量中山 根據次數增加 月工中山敏感度  
  
72.永遠握住阿津  
獲得方式 : 使用手部字體取得  
被動效果 : 手部字體等級 & 第一慾望獲得率 提升  
被動反效果 : 同一天中 給不同的阿津手部字體 開場第一慾望 提升  
  
73.聽見阿津的話題  
獲得方式 : 猥談攻擊時取得  
被動效果 : 第一慾望上限 提升  
被動正效果 : 同一場戰鬥中 根據猥談次數增加人生耐性  
  
74.對阿津毫無抵抗的洞口  
獲得方式 : 被洞部字體取得  
被動效果 : 每回合第一與第五慾望 & 洞口壓力量的量 提升  
被動反效果 : 同一天中 每被抽山 將提升洞口中山敏感度  
  
75.母豬的三口天堂  
獲得方式 : 數次被三個字體取得  
被動效果 : 開場所有慾望 & 被三個字體時爆擊率 提升  
  
76.月工100%  
獲得方式 : 月工字體時 100%取得  
被動效果 : 開場第一與第五慾望上限 提升  
被動反效果 : 同一天中 每被月工抽山 將提升月工敏感度  
  
77.失控的下半身  
獲得方式 : 數次被洞部&月工字體取得  
被動效果 : 當沒被二個字體時 第一慾望自動 提升 & 當被二個字體時 人生耐性 下降  
  
78.監獄人氣女典獄長  
獲得方式 : 數次使用or被洞部字體取得  
被動效果 : 洞部字體等級 & 第一慾望上限 提升  
被動反效果 : 同一場戰鬥中 每被洞口抽山 每回合開始第四慾望自動提升  
  
79.鬆弛的洞口 壓力的好去處  
獲得方式 : 數次被洞口中山取得  
被動效果 : 洞口敏感度 & 第一與第五慾望獲得率 提升  
被動反效果 : 同一天中 每被洞口中山 防禦力降低  
  
80.陷入100%地獄的廉價清潔工  
獲得方式 : 數次100%取得  
被動效果 : 100%時失神機率 下降  
被動反效果 : 同一天中 每100%時 所有耐性下降
