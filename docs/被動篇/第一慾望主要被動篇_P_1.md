台灣自製攻略 & Taiwan homemade Raiders

\--------------------  
請注意 被動順序依本人獲得為主  
1.禁斷入口  
獲得方式 : 第一次與囚犯戰鬥取得  
被動效果 : 每回合隨機慾望提升  
  
2.阿津獵人 > [施虐稱號篇](/稱號篇/施虐稱號篇.md)  
獲得方式 : 打贏囚犯數場取得  
被動效果 : 施虐等級 & 力量 % 第一慾望上限 提升  
  
3.阿津欺凌專家 > [施虐稱號篇](/稱號篇/施虐稱號篇.md)  
獲得方式 : 打贏囚犯數場取得  
被動效果 : 施虐等級 提升 & 當使用胯下踢擊時 第一慾望 獲取根據 施虐等級提升  
被動反效果 : 同一天中 每次使用胯下踢擊 更容易發生 興奮狀態  
  
4.渾身燥熱  
獲得方式 : 1.保持興奮模式戰鬥數場取得 2.第二層BOSS戰結束後取得  
被動效果 : 所有慾望上限 提升  
被動反效果 : 興奮狀態 下 所有慾望自動提升  
  
5.初見壓力量瞬間  
獲得方式 : 第一次看過囚犯釋放壓力量取得  
被動效果 : 精神 & 第一慾望上限 提升  
  
6.目睹阿津  
獲得方式 : 第一次看過囚犯使用阿津取得  
被動效果 : 精神 & 第一慾望上限 提升  
  
7.目睹不可言說之物  
獲得方式 : 數次看過囚犯使用阿津取得  
被動效果 : 第一慾望上限 提升  
新增技能 : 凝視阿津  
  
8.變身慾女的徵兆  
獲得方式 : 首次使用凝視阿津取得  
被動效果 : 第一慾望上限 提升  
技能升級 : 凝視阿津 效果 提升 & 冷卻時間 縮短  
  
9.對阿津的好奇心  
獲得方式 : 數次使用凝視阿津取得  
被動效果 : 開場第一慾望 提升  
技能升級 :  凝視阿津 效果 提升  
技能反效果 : 在同一場戰鬥中 每看過一隻阿津 第一慾望獲得提升  
  
10.觀察阿津愛好  
獲得方式 : 數次使用凝視阿津取得  
被動效果 : 每回合開始 根據上揚的阿津而提升 開始第一慾望  
技能反效果 : 在同一天中 每看一次阿津 當天開場第一慾望 提升  
  
11.猥談女主角  
獲得方式 : 戰鬥中聽囚犯說話  
被動效果 : 第一慾望上限 提升  
被動正效果 : 在同一天中 每聽到一次猥談 攻擊力提升  
  
12.躲藏的視野  
獲得方式 : 第一慾望 50時取得  
被動效果 : 第一慾望上限 提升  
被動反效果 : 第一慾望高於50時 攻擊力下降  
  
13.初次升天  
獲得方式 : 首次100%取得  
被動效果 : 所有慾望上限 提升 & 觸碰與人生耐性 下降  
被動反效果 : 在同一場戰鬥中 每次100%時 精力上限 下降  
  
14.100%餘溫  
獲得方式 : 100%時無法行動取得  
被動效果 : 每回合隨機慾望自動提升 & 100%時餘溫更多  
被動正效果 : 100%時耗費的精力減少  
  
15.被挑撥的好奇心  
獲得方式 : 看囚犯使用阿津取得  
被動效果 : 第一慾望上限 提升 & 戰鬥中自我防衛機率提升 & 囚犯使用阿津時所有慾望上升更快  
  
16.被沾汙的美麗肌膚  
獲得方式 : 首次被壓力量沾到身上取得  
被動效果 : 第一慾望上限 提升  
被動反效果 : 敏捷 根據壓力量在身上而 下降  
  
17.沾上壓力量的觸感  
獲得方式 : 數次被壓量沾到身上取得  
被動效果 : 第一慾望上限 提升 & 當使用 口&乳&足 字體時 體外釋放壓力量敏感度提升  
  
18.壓力量愛好  
獲得方式 : 數次被壓量沾到身上取得  
被動效果 : 第一慾望上限 提升  
被動反效果 : 落衣耐性根據身上的壓力量而下降  
  
19.被阿津吸引的目光  
獲得方式 : 數次看過囚犯使用阿津取得  
被動效果 : 第一慾望上限 提升 & 戰鬥中自我防衛機率提升 & 囚犯使用阿津時所有慾望上升更快  
  
20.對阿津的美妙幻想  
獲得方式 : 數次看過囚犯使用阿津取得  
被動效果 : 興奮狀態時 第一慾望自動提升  
被動反效果 : 第一慾望不是最高時 第一慾望獲得提升
