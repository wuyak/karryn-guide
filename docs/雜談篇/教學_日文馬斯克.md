\--------------------  
解決 日文馬斯克 問題  
請打開遊戲 Karryn's Prison 資料夾  
選擇 www 中的 js 之後點擊 plugins  
文件夾 裡面的 RemtairyDebugMode  
用 記事本 打開此文件  
往下拉到最下面  
在這行 return JAPAN\_MODE 前面加上 //  
變成 //return JAPAN\_MODE (懶人法直接複製貼上)  
保存之後進入遊戲 日文就沒有馬斯克了  
請用 v 1.0.1 G3 版本的 RemtairyDebugMode 套用到新版本 之前版本也可以喔  
之前 v 1.0.1 H4 版本 已失效 上面方法有效  
最新版本 v 1.0.1 K 一樣有效
